Tabs are created with <ul class="tab-links">:

Tip: Also to mark the current page active, add .active class inside the <a>tag.

The following example creates navigation tabs:
Example:
<div class="tabs">
 <ul class="tab-links">
  <li> <a class="active" href="#">Home </a></li>
  <li> <a href="#">Menu 1 </a></li>
  <li> <a href="#">Menu 2 </a></li>
  <li> <a href="#">Menu 3 </a></li>
  </ul>
</div>

You can also add a content on each tabs by using .tab-content
  • Home
  • Menu 1

  • Menu 3
Example:
<div class="tab-content">
 <p>Content</p>
</div>
