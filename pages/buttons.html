BUTTONS

Variations on Klasify buttons.
TO INCLUDE AN KLASIFY BUTTON COMPONENT:

1. Code a <button> element. Include any desired attributes and values, such as an id or event handler, and add a text caption or image as appropriate.

<button>Save</button>

2. Add one or more Klasify classes, separated by spaces, to the button using the class attribute.

<button class="button-orange color-white">Save</button>

The button component is ready for use.

EXAMPLES:
<button class="button-lime">Leonardo</button>
<button class="button-default color-white">Donatello</button>
<button class="button-orange color-white">Raphael</button>
<button class="button-white color-white">Michelangelo</button>
