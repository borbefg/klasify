Button Styles

Klasify provides four styles of buttons

To achieve the button styles above, Klasify has the following classes:

  • .button-lime
  • .button-default
  • .button-orange
  • .button-white

The following example shows the code for the different button styles:

EXAMPLES:
<button class="button-lime">Quas</button>
<button class="button-default color-white">Wex</button>
<button class="button-orange color-white">Exort</button>
<button class="button-white color-white">Invoke</button>

The button classes can be used on an <a>, <button>, or <input> element:

EXAMPLES:
<a href="#" class="button-lime">Link Button</a>
<button type="button" class="button-default color-white">Button</button>
<input type="button" class="button-orange color-white" value="Input Button">
<input type="submit" class="button-orange color-white" value="Submit Button">

Why do we put a # in the href attribute of the link?

Since we do not have any page to link it to, and we do not want to get a "404" message, we put # as the link. In real life it should of course been a real URL to the "Search" page.
Active/Disabled Buttons

A button can be set to an active (appear pressed) or a disabled (unclickable) state:

The class .active makes a button appear pressed, and the class .disabled makes a button unclickable:

EXAMPLES:
<button class="button-default color-white active">Active</button>
<button class="button-default color-white disabled">Disabled</button>
