Installation

You can install Klasify on your project by simply using a CDN or using Bower.

<link rel="stylesheet" src="https://bowercdn.net/c/klasify-0.4.4/stylesheets/klasify.css">
$ npm install -g bower

$ bower install klasify --save

At Klaseko we use FortAwesome's Font Awesome icon set. We haven't designed our own custom icon set (yet!) Our CSS framework goes well with this icon set so we recommend you import it to your project as well.
